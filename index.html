五个任务汇总入口

这里集中访问 Task 1 到 Task 5 的 HTML 成果页。所有分析均使用课程数据和复权价格口径，仅用于学习研究，不构成投资建议。

Task 1

中际旭创股价分析

复权价格、K 线、回撤和波动率分析。

 Task 2

Indicator Lab

RSI、MACD、布林带、ATR、MA、KDJ、OBV 交互式指标实验台。

 Task 3

双均线策略回测

5/15 日均线信号、模拟交易、绩效对比和参数敏感性。

 Task 4

海龟策略回测

唐奇安通道、ATR/N 值、2N 止损、风险定仓和多标的回测。

 Task 5

股票分类机器学习

逻辑回归、决策树、随机森林、KNN，以及混淆矩阵、ROC 与 AUC 交互评估。
