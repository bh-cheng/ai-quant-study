# 多元线性回归分析工具

这是一个无需后端的静态网页工具。CSV 的解析、OLS 计算、诊断、绘图与导出全部在浏览器本地完成。

## 使用方式

通过任意静态文件服务器打开本目录的 `index.html`，或从项目根目录汇总页进入。页面支持上传 CSV、粘贴 CSV 和内置示例数据。

CSV 第一行必须为唯一字段名。首版只允许数值列作为 Y/X；缺失值会按当前所选变量整行剔除并报告。

## 文件说明

- `index.html`：工具页面结构。
- `styles.css`：响应式和打印样式。
- `app.js`：导入、交互、结果表格与导出。
- `csv.js`：CSV 解析、分隔符识别和字段类型推断。
- `stats.js`：带列主元的 Householder QR、OLS 与统计诊断。
- `charts.js`：六类 Canvas 诊断图。
- `sample-data.csv`：可下载/复用的业务经营示例。
- `tests/`：统计与 CSV 单元测试。

工具不依赖 CDN 或第三方运行时库，也不会发起遥测或分析请求。

## 本地校验

如需运行随附的统计测试，可在本目录执行 `npm test`。测试不安装任何依赖。

本工具只提供统计分析辅助。线性关系、独立性、同方差、残差分布和研究设计仍需结合实际业务判断。
